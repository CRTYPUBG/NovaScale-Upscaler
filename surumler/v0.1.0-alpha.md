# NovaScale v0.1.0-alpha

**Etiket:** `v0.1.0-alpha`  
**Yayın Başlığı:** Initial Test Build (Alpha)

## Sürüm Notları
Bu, NovaScale'in ilk halka açık test sürümüdür. Temel capture ve upscale mekanizmalarının kararlılığını test etmeyi amaçlar.

### Özellikler:
- **DXGI Desktop Duplication:** Düşük gecikmeli ekran yakalama.
- **Bilinear Upscaling:** Temel uzamsal ölçeklendirme (Placeholder).
- **Borderless Fullscreen Output:** Oyunun üzerinde şeffaf olmayan üst katman sunumu.
- **Anti-Cheat Safe:** Herhangi bir DLL injection veya API hooking içermez.
- **Gelişmiş Log Sistemi:** Çökme durumunda otomatik hata raporlama ve log açma.

### Bilinen Kısıtlamalar:
- Henüz FSR veya CNN algoritmaları aktif değildir.
- Sadece Borderless veya Windowed modda çalışan oyunları destekler.
- Çoklu monitör desteği alpha aşamasında sınırlıdır.
